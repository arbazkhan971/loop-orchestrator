# Branding

Loop Orchestrator is positioned as an open-source tmux AI agent team orchestrator.

## Tagline

Open-source tmux AI agent teams for Claude Code, Codex, Gemini CLI, and custom terminal coding agents.

## Short Description

Loop Orchestrator runs configurable AI software engineering teams inside tmux: planner, frontend, backend, QA, scout, and release roles with local auth detection, prompt generation, and a dashboard.
